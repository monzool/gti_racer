## Installation

Copy *git_racer.sh* to somewhere and make it executable

```sh
chmod +x /path/to/git_racer.sh
```

Add an alias in your .bashrc/.zshrz/.profile or similar

```sh
alias gti=/path/to/git_racer.sh
```

## Usage

```sh
gti
```
